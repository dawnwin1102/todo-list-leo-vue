# 📸 应用截图

本目录包含 Todo 应用的实际截图，展示迪士尼风格的界面设计。

## 📋 截图列表

- `dashboard.png` - Dashboard 仪表板页面截图
- `todo-management.png` - 任务管理页面截图  
- `ui-details.png` - 界面细节特写截图

## 🎯 生成截图

使用以下命令生成新的截图：

```bash
# 自动生成截图
node screenshots/capture.js

# 或使用手动脚本
bash screenshots/capture.sh
```

## 📝 截图说明

所有截图都是使用 Playwright 自动生成的，真实展示了应用的运行效果。截图展示了：

- 🏰 梦幻渐变背景和玻璃态效果
- 📊 数据可视化图表和统计卡片
- 🎨 流畅的动画和交互效果
- 📱 响应式设计适配
